## 概要

USJ公式アプリの使いづらさを改善したアプリのデザインを作成しました。

### 使いづらさのポイント
> ホーム画面にアイコンが沢山あることでユーザが困惑する<br>
> ユーザが一番利用する待ち時間がUSJに入園しないと見られない

<br>
このあたりを改善したいのでホームをユーザによってカスタマイズできるように構想しました。<br>
さらに、ホームに沢山あったアイコンはメニューに統一しました。
